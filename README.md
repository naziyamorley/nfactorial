Chessy — платформа, нацеленная на развитие шахматного спорта в Казахстане через современные игровые технологии и сильное комьюнити. Проект делает шахматы более интересными и доступными для школьников, студентов и начинающих игроков благодаря RPG-механикам, AI-тренеру и интерактивному обучению. Среди главных фич — онлайн-дуэли, AI-анализ партий, турниры, система прокачки героев, школьный модуль для учителей и учеников, а также карта шахматных клубов и соревнований по всему Казахстану.
